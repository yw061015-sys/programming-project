# Game Review Platform - 최종 기획안

## 📋 프로젝트 개요

**프로젝트명**: Game Review Platform  
**목표**: 게이머들이 게임 리뷰를 공유하고, 커뮤니티와 상호작용하며, 최신 게임 정보를 얻을 수 있는 통합 플랫폼  
**기술 스택**: React 19, Tailwind CSS 4, TypeScript, Wouter (라우팅)  
**배포 환경**: Manus 웹 플랫폼  
**배포 URL**: https://3000-ipaubvzhar0c0jcu15xmp-f9f45a3a.sg1.manus.computer

---

## 🎯 핵심 기능

### 1. 인증 시스템 (로그인/로그아웃)
- **기능**: localStorage 기반 간단한 인증
- **테스트 계정**: admin / password
- **특징**: 
  - 로그인하지 않으면 메인 페이지 접근 불가
  - 로그인 상태 유지 (localStorage)
  - 로그아웃 기능

### 2. 게임 리뷰 작성 및 관리
- **기능**: 게임 리뷰 작성, 저장, 표시
- **입력 필드**:
  - 게임 제목 (필수)
  - 플랫폼 선택 (PC, PlayStation, Xbox, Nintendo Switch, Mobile)
  - 평점 (1~5점)
  - 리뷰 내용 (상세한 의견)
  - 게임 스크린샷 (선택사항 - 이미지 첨부)
- **특징**:
  - 최신 리뷰가 상단에 표시
  - 이미지는 Base64로 인코딩되어 저장
  - localStorage에 모든 데이터 저장

### 3. 커뮤니티 채팅
- **기능**: 실시간 메시지 기반 커뮤니티
- **특징**:
  - 메시지 입력 및 전송
  - 자신의 메시지는 보라색 그라디언트로 표시
  - 다른 사용자의 메시지는 회색으로 표시
  - 모든 메시지는 localStorage에 저장
  - 채팅방 입장 버튼으로 쉬운 접근

### 4. 최신 게임 뉴스
- **기능**: 게임 뉴스 카드 및 상세 페이지
- **뉴스 항목**:
  - 2025년 기대작 게임 TOP 10
  - PS5 Pro 공식 출시
  - 인기 RPG 시리즈 신작
  - e스포츠 월드 챔피언십
- **특징**:
  - 각 뉴스 카드는 클릭하면 상세 페이지로 이동
  - 요약, 상세 내용, 주요 정보 포함
  - 뉴스 날짜 표시

### 5. 게임 가이드
- **기능**: 게임 공략 및 팁 제공
- **가이드 항목**:
  - 젤다의 전설: 티어스 오브 더 킹덤
  - 엘든 링
  - 스타듀 밸리
- **특징**:
  - 아코디언 UI로 펼쳐지는 구조
  - 각 게임별 상세한 공략 정보

### 6. 사용자 프로필
- **기능**: 프로필 정보 관리 및 활동 내역 확인
- **프로필 정보**:
  - 사용자명
  - 이메일
  - 자기소개
  - 프로필 정보 수정 기능
- **활동 내역**:
  - 작성한 리뷰 목록 (0개)
  - 커뮤니티 메시지 목록 (1개)
  - 평균 평점 (0.0)
- **특징**:
  - 탭 기반 인터페이스 (프로필 정보, 작성한 리뷰, 커뮤니티 활동)
  - 정보 수정 시 localStorage에 저장

│   │   │   ├── NewsSection.tsx    # 뉴스 섹션
│   │   │   ├── GuideSection.tsx   # 가이드 섹션
│   │   │   └── (UI 컴포넌트)

│   │   ├── index.css              # 글로벌 스타일
│   │   └── main.tsx               # 진입점
│   └── index.html

- **ReviewSection**: 리뷰 작성 폼, 리뷰 목록, 이미지 첨부
- **CommunitySection**: 커뮤니티 채팅방 입장 버튼
- **NewsSection**: 뉴스 카드 목록, 상세 페이지 링크
- **GuideSection**: 아코디언 기반 게임 가이드
- **Profile**: 프로필 정보, 활동 내역, 수정 폼
