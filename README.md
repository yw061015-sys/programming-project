## 🎯 프로젝트 목표

게이머들이 게임 리뷰를 작성하고, 커뮤니티에서 소통하며, 최신 게임 뉴스를 확인하고, 게임 공략을 얻을 수 있는 **통합 게임 리뷰 및 커뮤니티 플랫폼**을 제공합니다.

---

## ✨ 핵심 기능 (7가지)

### 1️⃣ 인증 시스템
- **로그인**: 기존 사용자 로그인
- **회원가입**: 새로운 사용자 등록
  - 아이디 중복 확인
  - 이메일 형식 검증
  - 비밀번호 일치 확인
  - 6자 이상 비밀번호 요구
- **로그아웃**: 세션 종료
- **localStorage 기반**: 클라이언트 사이드 인증

### 2️⃣ 게임 리뷰 시스템
- **리뷰 작성**: 게임 제목, 플랫폼, 평점(1-5), 상세 의견 입력
- **이미지 첨부**: 게임 스크린샷 업로드 (Base64 인코딩)
- **리뷰 목록**: 최신순으로 정렬된 리뷰 표시
- **최신 리뷰 강조**: 가장 최근 리뷰를 보라색 그라디언트로 표시
- **평점 시각화**: 별 아이콘으로 평점 표시
- **localStorage 저장**: 페이지 새로고침 후에도 데이터 유지

### 3️⃣ 실시간 커뮤니티 채팅
- **채팅 인터페이스**: 메시지 입력 및 전송
- **사용자 구분**: 자신의 메시지(보라색), 다른 사용자(회색)
- **메시지 타임스탬프**: 작성 시간 표시
- **실시간 업데이트**: 메시지 즉시 표시
- **localStorage 저장**: 채팅 히스토리 유지

### 4️⃣ 최신 게임 뉴스
- **뉴스 카드**: 4개의 Mock 뉴스 데이터
- **뉴스 상세 페이지**: 각 뉴스 클릭 시 상세 정보 표시
  - 뉴스 제목, 요약, 상세 내용
  - 발행일, 카테고리
  - 뉴스 이미지
- **자세히 보기**: 상세 페이지로 이동하는 버튼

### 5️⃣ 게임 가이드
- **아코디언 UI**: 게임별 가이드 펼치기/접기
- **3개 게임 가이드**: 젤다의 전설, 엘든 링, 스타듀 밸리
- **상세 공략**: 각 게임별 팁과 전략 정보

### 6️⃣ 사용자 프로필
- **프로필 정보**: 사용자명, 이메일, 자기소개
- **프로필 수정**: 정보 업데이트 기능
- **활동 통계**: 작성한 리뷰 수, 커뮤니티 메시지 수, 평균 평점
- **커뮤니티 활동**: 사용자가 작성한 메시지 목록 표시

### 7️⃣ 네비게이션 및 UI
- **상단 네비게이션**: 📝 리뷰, 💬 커뮤니티, 📰 뉴스, 🎯 가이드 버튼
- **섹션 스크롤**: 각 버튼 클릭 시 해당 섹션으로 부드럽게 이동
- **헤더**: 사용자명 표시, 프로필 버튼, 로그아웃 버튼
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
